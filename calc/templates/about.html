О проекте

Этот проект представляет собой простой веб-калькулятор, созданный на языке Python с использованием фреймворка Django.

Пользователь может вводить математические выражения, получать результат и просматривать историю выполненных операций.

История операций сохраняется в базе данных SQLite. Для каждой сессии пользователя создаётся отдельная история.

В проекте используются отдельные файлы для HTML, CSS и JavaScript. Это делает структуру сайта более правильной и понятной.
